Concept for a public square artwork: This dome type object provides shade to an otherwise empty courtyard, and can be outfitted with misting water sprayers to help with the summer heat. At night, the colorful panels glow from within, creating an ambient atmosphere and providing safety to onlookers. This system can run on solar power and batteries, and be a power-positive system; producing more power than it consumes.

 

 

 

Mural conceptual artwork: This row home stands out from its surroundings, creating a center of attention that can bring an audience. This design would be great for an arts center, public learning place, library, or any other building that has to be noticed in the crowded streets.

 

 

 

Concept for Public Sculpture: These arches frame the area around them, and create a central point of interest and landmark. Scalable, they can be adapted to almost any size, and compliment their surroundings depending on the color choice for the clear panels. At night, these arches glow from within.

 

 

 

Mural Concept: This building appears to have been painted in an instant! A fun concept, great for a fun atmosphere within. An arts organization, or anyone who wants to garner attention, would be pleased by this mural. This concept is scalable, able to accomidate any size structure.
